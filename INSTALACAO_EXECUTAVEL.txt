🚀 INSTALAÇÃO EM OUTRO COMPUTADOR
================================================================================

PASSO 1: Copiar o Executável
-----------------------------
1. Copie o arquivo "Controle_DDU.exe" que está na pasta "dist"
2. Cole em qualquer pasta no computador de destino
   Sugestão: C:\Programas\Controle_DDU\ ou Desktop

PASSO 2: Primeira Execução
---------------------------
1. Dê um duplo clique no arquivo "Controle_DDU.exe"
2. O Windows pode exibir um aviso de segurança (SmartScreen)
   - Clique em "Mais informações"
   - Depois clique em "Executar assim mesmo"
3. Na primeira execução, o programa criará automaticamente:
   - O banco de dados SQLite (documents.db)
   - Todas as tabelas necessárias

PASSO 3: Criar Atalho (Opcional)
---------------------------------
1. Clique com botão direito no "Controle_DDU.exe"
2. Selecione "Enviar para" > "Área de trabalho (criar atalho)"
3. Agora você pode executar o programa diretamente da área de trabalho

📂 ARQUIVOS CRIADOS PELO PROGRAMA
================================================================================
O programa cria automaticamente os seguintes arquivos na mesma pasta do executável:

1. documents.db
   - Banco de dados SQLite com todos os documentos cadastrados
   - É criado automaticamente na primeira execução
   - Contém todas as informações dos documentos, DDUs, etc.

2. pmmg.png (se fornecido)
   - Logo da PMMG usado no cabeçalho
   - Já está embutido no executável


🔄 BACKUP E MIGRAÇÃO DE DADOS
================================================================================

Para fazer backup dos dados:
----------------------------
1. Copie o arquivo "documents.db" que está na mesma pasta do executável
2. Guarde em local seguro (pen drive, nuvem, etc.)

Para restaurar dados em outro computador:
-----------------------------------------
1. Instale o Controle_DDU.exe no novo computador
2. Execute o programa uma vez para criar a estrutura
3. Feche o programa
4. Substitua o arquivo "documents.db" pelo seu backup
5. Abra o programa novamente - todos os dados estarão lá!

🔧 SOLUÇÃO DE PROBLEMAS
================================================================================

Problema: O programa não abre
-----------------------------
Solução:
- Verifique se o arquivo está corrompido (tamanho deve ser ~55 MB)
- Desabilite temporariamente o antivírus
- Execute como Administrador (botão direito > "Executar como administrador")

Problema: Erro ao gerar PDF
----------------------------
Solução:
- Certifique-se de que há documentos cadastrados
- Para PDF Positivo, certifique-se de que há documentos com checkbox marcado
- Verifique se você tem permissão de escrita na pasta de destino

Problema: Dados não aparecem
-----------------------------
Solução:
- Verifique se o arquivo "documents.db" está na mesma pasta do executável
- Se moveu o executável, mova também o arquivo "documents.db"
